# Solar Panel Defect Detection using Computer Vision

This project focuses on detecting defects in solar panels using a CNN model built with PyTorch and OpenCV.

## Overview
The goal of this project is to classify solar panel images as either:
- GOOD
- DEFECT

The model was trained on a custom dataset of solar panel images.

## Features
- Image preprocessing using OpenCV
- CNN-based image classification
- Confidence-based predictions
- Uncertainty handling
- Training and testing pipeline

## Technologies
- Python
- PyTorch
- OpenCV
- NumPy

## Results
The model is able to distinguish between good and defective solar panels with approximately 60–70% accuracy on test data.

## Future Improvements
- Improve dataset quality
- Use pretrained CNN architectures
